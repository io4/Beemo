# Beemo 

A multi-purpose bot for discord. [Visit my site](https://beemo.club)

###Beemo uses and loves
- Discord (duh)
- Redis
- Discord.js
- Wolfram|Alpha
- Cleverbot
- Express
- Socket.io

###Installation

```bash
npm install
cp credentials.json.example credentials.json
nano credentials.json #edit it

node --harmony app.js
```
